# FurnitureLand
Managers of Furniture Land can use this mobile application.

**Functionalites of Furniture Land mobile application**

1.Login

2.Create an account

3.Manage Employees

4.Manage Products

5.Manage Categories

6.Manage Subcategories

7.Manage Newsfeed

8.View Photo Gallery
